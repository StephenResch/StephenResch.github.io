Purpose

There are two main purposes for this website. The first is for having a
place to put my game, because in its current state, it's size does not
validate a spot on Steam, a PC gaming platform. The second reason for creating this website is to keep followers of the game updated on whatever is going on. This can include game updates, Q&A's and community polls.
